Sort algorithms
###############

Quicksort
=========

Time complexity
---------------

*   Worst case performance: :math:`O(n^2)`
*   Best case performance: :math:`O(n\log n)`
*   Average case performance: :math:`O(n\log n)`

Space complexity
----------------

*   Worst case space:

    *   :math:`O(n)` (naive)
    *   :math:`O(\log n)` (in-place)
