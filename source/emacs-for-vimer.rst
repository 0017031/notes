Emacs for Vimer
###############

Check `this incomplete cheatsheet`__.

``gf`` (goto file)
    ``M-x ffap`` / ``find-file-at-point``.  Add this line to ``.emacs`` to enable ``ffap`` bindings::

        (ffap-bindings)

    Then put cursor on a file name, enter ``C-x C-f``.
